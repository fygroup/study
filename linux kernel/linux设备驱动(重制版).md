### io 体系
```
CPU和IO设备之间的数据通路通常称为IO总线

总线分类：地址总线、数据总线、控制总线

每个IO设备依次连接到总线上，这种连接使用了包含3个元素的硬件组织层次：IO端口、接口、设备控制器

> IO端口
    每个连接到IO总线上的设备都有自己的IO地址集，通常称为IO端口
    
    有4条专用的汇编语言指令可以允许CPU对IO端口进行读写，它们是in、ins、out和outs
    
    在执行其中的一条指令时，CPU使用'地址总线'选择所请求的IO端口，使用'数据总线'在CPU寄存器和端口之间传送数据

    IO端口既可以使用独立的IO地址空间，也可以被映射到物理地址空间，处理器和IO设备之间的通信就可以使用对内存直接进行操作的汇编指令

         -> 控制寄存器 ->
    cpu  <- 状态寄存器 <-  设备IO接口  
         <- 输入寄存器 <-
         -> 输出寄存器 ->

> IO接口


```
